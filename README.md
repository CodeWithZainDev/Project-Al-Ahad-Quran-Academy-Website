# Project Al-Ahad Quran Academy Website

A modern and responsive website developed for **Al-Ahad Quran Academy**. This project is designed to provide an online presence for the academy, allowing students and parents to learn about available courses, academy services, instructors, and contact information.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Clean and user-friendly interface
* Course and program information
* Contact and inquiry section
* Structured navigation for easy browsing
* Modern web design practices

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Goals

The primary goal of this project is to create a professional website that helps Al-Ahad Quran Academy:

* Showcase its educational services
* Provide information about Quran and Islamic studies courses
* Improve communication with students and parents
* Establish a strong online presence

## Learning Outcomes

Through this project, I practiced:

* Semantic HTML structure
* Responsive web design
* CSS styling and layouts
* Website organization and user experience principles
* Front-end development best practices
